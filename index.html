    Web Pixel UI Engine (WPUIE)

    # About

    WPUIE 皆在为像素风格网页或像素游戏提供组件。

    # Installation

    目前仅有css文件，你可以点击此处下载。未来会加入js脚本以提供更丰富的操作。当然你可以仅仅使用css。
    如果你没有更丰富的定制需求(例如页面中不同大小和颜色的border)，你可以使用NES.css，其对网络更为友好。
